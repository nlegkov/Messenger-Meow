# Messenger-Meow

Meow Messenger — учебный проект по дисциплине «Цифровые платформы», представляющий собой простой мессенджер для обмена текстовыми сообщениями в локальной сети.

Проект реализуется на базе процессора CDM-8E, смоделированного в Logisim, и использует интерфейс UART для последовательной передачи данных между устройствами.

⸻

Overview

Цель проекта — продемонстрировать принципы работы цифровых платформ и взаимодействия вычислительных устройств через последовательный интерфейс.

В системе реализована передача сообщений между несколькими узлами сети.
Каждый узел представляет собой цифровую систему с процессором CDM-8E, выполняющим программу обработки и передачи сообщений.

Основные функции системы:
	•	отправка текстовых сообщений
	•	прием сообщений от других устройств
	•	передача данных через UART
	•	работа в пределах локальной сети

⸻

Features
	•	текстовый обмен сообщениями между узлами сети
	•	реализация логики на уровне цифровых схем
	•	использование процессорной архитектуры CDM-8E
	•	последовательная передача данных через UART
	•	моделирование системы в Logisim

⸻

System Architecture

Система состоит из нескольких основных компонентов.

CDM-8E Processor

Центральный компонент системы.
Процессор выполняет программу мессенджера и управляет обработкой входящих и исходящих сообщений.

UART Communication Module

Модуль последовательной передачи данных.
Используется для обмена сообщениями между устройствами через интерфейс UART.

Input Module

Обеспечивает ввод сообщений пользователем.

Output Module

Отвечает за отображение полученных сообщений.

⸻

Requirements

Для запуска проекта требуется:
	•	Logisim (или Logisim Evolution)
	•	схема процессора CDM-8E
	•	несколько экземпляров системы для обмена сообщениями

⸻

Running the Project
	1.	Установить Logisim.
	2.	Открыть схему системы из каталога logisim/.
	3.	Запустить симуляцию.
	4.	Подключить несколько экземпляров системы через UART.
	5.	Ввести сообщение и отправить его другому узлу сети.
